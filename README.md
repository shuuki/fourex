# fourex

flying sim

| Keyboard     | Controls |
| W / S        | Power Forward/Back |
| A / D        | Slide Left/Right |
| R / T        | Slide Up/Down |
| Up / Down    | Pitch Forward/Back |
| Q / E        | Roll counterclockwise/Clockwise |
| Left / Right | Yaw Left/Right |
| Space        | Reset Ship |

Play: https://shuuki.github.io/fourex/fourex.html
